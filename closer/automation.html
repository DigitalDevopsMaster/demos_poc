Configuración de disparadores

Define cuándo se iniciará este flujo.

Configuración del disparador: Mensaje recibido
Separa las palabras con comas. El flujo se activará si el mensaje contiene alguna de estas palabras.
Configuración del disparador: Contacto creado
Esperar este tiempo después de la creación del contacto antes de activar el flujo.
Configuración del disparador: Etiqueta añadida
El flujo se activará cuando cualquiera de estas etiquetas sea añadida a un contacto.
Configuración del disparador: Formulario enviado
El flujo solo se activará si este campo fue completado en el formulario.
Configuración del disparador: Programado
El flujo se ejecutará solo para los contactos en este segmento.
Configuración del disparador: API / Webhook

Este flujo puede ser iniciado a través de una llamada a la API o webhook.

URL del Webhook:
https://api.closer.digital/webhook/flow/5f8d43a1b2e3c42a9f8d7e6b
Ejemplo de JSON:
{
  "contact_id": "c123456789",
  "data": {
    "name": "Juan Pérez",
    "email": "juan@ejemplo.com",
    "custom_field": "valor personalizado"
  }
}
El webhook fallará si falta alguno de estos campos.
